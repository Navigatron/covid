About this Map

This map displays covid-19 case-counts, by county, on a logarithmic color scale. No alarmism (red on black) or silly circles.

A logarithmic scale displays detail in small ranges (1 and 5 are distinguishable), while also showing percentage changes in the thousands.

Human eyes see certain colors as 'brighter' than others. This map's color scale uses the HCL (Hue, Chroma, Percieved Lightness) model to compensate for this problem.

As a result this map is color-blind friendly, easy on the eyes, and clearer to interpret.

Read more about better color scales for scientific graphics on betterfigures.org.
