About this Map

This map displays covid-19 case-counts, by county, on a logarithmic color scale.

A logarithmic scale displays detail in small ranges (1 and 5 are distinguishable), while still being able to show changes in the thousands.

Human eyes see certain colors as brighter than others. This map uses the HCL (Hue, Chroma, Percieved Lightness) model to compensate for this problem.

Read more about better color scales for scientific graphics on betterfigures.org.

Data is aggregated from multiple sources, including John Hopkins and the New York Times.
